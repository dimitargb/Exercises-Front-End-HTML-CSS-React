DOM манипулация:

DOM позволява на JavaScript да променя HTML елементите и това наричаме DOM Manipulations.

Това става посредством много на брой различни методи и пропертита като например:
- innerHTML - променя директно цялата HTML структура, може да добавя тагове и т.н. - а на деден елемент.
- textContent - променя текста
- value - използва се за input полетата.
- style - променя CSS
