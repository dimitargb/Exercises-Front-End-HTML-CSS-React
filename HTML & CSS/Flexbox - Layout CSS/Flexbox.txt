Flexbox - е система за разпределяне на пространството в кутийките като правиме и layout в нашия сайт.

Flex Container - parent на самите елементи.
Flex Items - самите елементи.

За да работим с Flexbox трябва да кажем display: flex - това прави всички деца на контейнера - Flex Items.
В контейнера имаме Main axis - основна координата - Cross axis - напречна координата.
Тук имаме main start и main end, както и cross start и cross end.

Main axis - това е на дължина и широчина - по хоризонтала.
Cross axis - това е на височина - по вертикала.
